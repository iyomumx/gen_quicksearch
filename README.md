#### 简介
这是一个简单的Winamp搜索插件，用于搜索并播放当前播放列表中的文件。
#### 操作说明
* 插件安装后，会注册一个全局热键设置项，可以在Winamp的“全局热键”设定中找到并绑定全局热键。
* 按下全局热键后会显示搜索窗口并自动激活`(Active)`并使输入焦点指向搜索文本框`(Focus)`。在文本框内的所有输入会被视为一个整体，同时对文件路径和标题（在播放列表编辑器中显示的文本）进行不区分大小写的匹配。
* 所有匹配项将显示在文本框下方的列表中，按播放列表中的顺序排序。
* 在文本框中按下Enter，则直接播放第一项。按下Tab则在下方列表中选中列表第一项。
* 当输入焦点在列表框时，按下Tab选择下一项。当选中最后一项时，会循环选择第一项。
* 在列表框中按下Q键则将选中项送入播放队列。（相当于在播放列表编辑器中选中此项并按下Q键）
* 双击列表框或在列表框中按下Enter将直接播放选中项。
* 当搜索窗口不再处于激活状态时`(OnDeactivated)`或任一项被播放或队列时，窗口隐藏。
* 搜索窗口不会保持上次显示时的搜索状态，而是会将搜索文本框清空以便进行下一次搜索。
